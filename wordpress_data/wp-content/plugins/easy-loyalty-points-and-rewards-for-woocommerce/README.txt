== Description ==

**Easy Loyalty Points and Rewards for WooCommerce** is a lightweight, easy to use customer loyalty system for WooCommerce. Increase customer loyalty by rewarding customers with points for purchases and actions. Points can be redeemed for discounts on future orders.

= FEATURES =

* **Earn points for purchases** - earn points at a conversion ratio you define.
* **Redeem points for discounts** - redeem points for discounts on future order at a conversion ratio you define.
* **Redemption thresholds** - define minimum order amounts or point balances before points can be redeemed for a discount.
* **Customer dashboard** - customers can view their points balance and recent points activity within their WooCommerce account page.
* **Manually adjust points balances for customers** - adjust balances at any time.
* **Custom messages** - choose your own custom messages to be displayed to customers (although sensible defaults are included).
* **Points log** - every points activity is recorded in a searchable global log.
* **Customer balances** - view a list of customers with their points balances.
* **Tax/VAT mode** - award points based on tax inclusive or tax exclusive prices.
* **Point rounding** - round points up or down.

== Installation ==

1. Unzip `easy-loyalty-points-and-rewards-for-woocommerce.zip` and upload to the `/wp-content/plugins/` directory, or install the plugin through the **Plugins** page in WordPress
2. Activate the plugin through the **Plugins** menu in WordPress
3. Go to **WooCommerce > Points and Rewards > Settings** to configure the plugin settings

== Frequently Asked Questions ==

= How does this plugin handle tax/VAT? =

The plugin can calculate using tax inclusive or tax exclusive prices. This can be changed using the plugin's 'Tax Mode' setting.

= Does the plugin support import/export of point balances? =

Not yet but this is planned for the future.

= Does the plugin support assigning points for orders created before the plugin was installed? =

Not yet but this is planned for the future.
